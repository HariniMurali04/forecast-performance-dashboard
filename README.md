# forecast-performance-dashboard
Financial Forecast Performance Dashboard using Power BI, Power Query, and DAX



## Overview
Developed an interactive Power BI dashboard to compare forecast and actual performance across Revenue, Non-Revenue, and CAPEX.

## Features
- Forecast Accuracy KPI
- Forecast Bias KPI
- Dynamic Lag Selection
- Country Filter
- Period Filter
- Executive Dashboard

## Tools Used
- Power BI
- Power Query
- DAX

## Key Skills
Data Analytics, Business Intelligence, Dashboard Development, Data Modeling
